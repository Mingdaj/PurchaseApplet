# PurchaseApplet
全天候购药系统（微信小程序+web后台管理）

传统线下购药方式存在无法全天候向用户提供购药服务，无法随时提供诊疗服务等问题。为此，运用软件工程开发规范，充分调研建立需求模型，编写开发文档，实现了一款全天候一般疾病购药系统。
